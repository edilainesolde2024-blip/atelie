# The Digital Atelier

Landing page moderna e responsiva desenvolvida com HTML e CSS.

## ✨ Sobre o Projeto

O projeto **The Digital Atelier** é uma interface web inspirada em layouts premium modernos, focada em:

* Design elegante
* Responsividade
* Interações suaves
* Estrutura limpa
* Experiência visual moderna

A página possui:

* Header fixo
* Hero section estilizada
* Imagem destaque
* Card flutuante de depoimento
* Botões interativos
* Cards informativos
* Responsividade para mobile
* Efeitos hover e animações leves

---

# 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts
* Material Symbols

# 🚀 Funcionalidades

## ✅ Layout Responsivo

A interface se adapta para:

* Desktop
* Tablet
* Mobile

Utilizando:

```css
@media (max-width: 768px)
```

---

## ✅ Header Dinâmico

Ao rolar a página:

* O header ganha blur
* Fundo translúcido
* Shadow suave

---

## ✅ Botões Interativos

Os botões possuem:

* Hover animation
* Transições suaves
* Efeito de elevação

---

## ✅ Cards Animados

Os cards sobem levemente ao passar o mouse.

---

## ✅ Scroll Suave

Implementado com:

```css
html {
  scroll-behavior: smooth;
}
```
